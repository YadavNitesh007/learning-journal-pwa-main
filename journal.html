Weekly Journal

Week 2 – Fundamentals of Responsive Web Design

1. How did you approach mobile-first design?

I began my design process with a mobile-first approach, utilizing a single-column layout, intuitive navigation, and responsive text sizes. I then implemented media queries to optimize the layout for tablets and desktop screens, ensuring a seamless user experience across devices.

2. What was the most useful HTML or CSS concept you applied this week?
The most valuable concept I applied this week was using Flexbox and Grid to create flexible, well-structured layouts. Additionally, implementing responsive CSS and effective navigation between pages helped me better understand overall page structure and user experience.

3. What part of HTML or CSS did you find most challenging?
The most challenging aspect was mastering responsive design with media queries and organizing project files in a clean, maintainable structure. Learning to integrate version control through GitHub was also new, but I was able to grasp it gradually through practice.
